|IDM_s| includes a command-line interface (CLI) with options and commands to assist with getting started, managing and monitoring, and troubleshooting simulations and experiments. After you've installed |IDM_s| you can view the available options and commands by typing the following at a command prompt

.. command-output:: idmtools --help
