CLI Slurm Platform
==================

The |SLURM_s| platform related commands can be accessed with ``idmtools slurm``. All slurm commands require job directory. See the details of the top level command below for detailed help:

.. command-output:: idmtools slurm <JOB_DIRECTORY> --help
   :returncode: 0
