=================
idmtools.ini File
=================

You can use the ``config`` command to create a configuration block in your project's ``idmtools.ini`` file.

``--config_path`` argument: If you do not specify a config path, the command will use the ``idmtools.ini`` file in the current directory.
To edit a different file, you can specify its path like ``idmtools config --config_path C:\my_project\idmtools.ini``


Use the ``block`` command to start the wizard that will guide you through the creation of a configuration block in the selected ``idmtools.ini``.
For example: ``idmtools config block``.
